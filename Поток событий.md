#Поток событий

##Начало игры в матче

Данный вариант использования позволяет пользователю подключиться к матчу и начать в нём действия
Предусловие: игра находится в состоянии ожидания нажатия одной из клавиш в главном меню
Основной поток событий
 
1. Вариант использования начинается, когда пользователь нажимает кнопку Start в главном меню.

2. Главное меню сменяется на экран загрузки.

3. Серверу отправляется запрос на подключение. При слишком долгом ожидании ответа выполняется альтернативный поток событий E1. 

4. При положительном ответе экран загрузки сменяется меню выбора персонажа.

5. При смене пользователем игрового персонажа меню перерисовывается с новыми параметрами.

6. При выборе персонажа, на сервер отправляется сигнал о готовности. При слишком долгом ожидании ответа выполняется альтернативный поток событий E1. 

7. При положительном ответе меню выбора персонажа сменяется игровой картой с фиксацией камеры на персонаже.

8. При нажатии кнопки перемещения игровая карта перерисовывается с новыми параметрами, а сообщение с изменениями отправляется серверу.

9. При нажатии кнопки взбирания на препятствия игровая карта перерисовывается с новыми параметрами, а сообщение с изменениями отправляется серверу.

10. При нажатии кнопки взаимодействия с объектами на карте игровая карта перерисовывается с новыми параметрами, а сообщение с изменениями отправляется серверу.

11. Сигнал синхронизации постоянно отправляется серверу, а полученный ответ с изменениями распаковывается и игровая карта перерисовывается с новыми параметрами. При слишком долгом ожидании ответа выполняется альтернативный поток событий E1.

12. Вариант использования завершается

##Изменение графических настроек

Данный вариант использования позволяет пользователю изменить графические настройки игры
Предусловие: игра находится в состоянии ожидания нажатия одной из клавиш в главном меню
Основной поток событий
 
1. Вариант использования начинается, когда пользователь нажимает кнопку Options в главном меню.

2. Главное меню сменяется на меню настроек.

3. При изменении параметра разрешения экрана, новые настройки сохраняются, а меню перерисовывается с новыми параметрами.

4. При изменении параметра ограничения FPS, новые настройки сохраняются, а меню перерисовывается с новыми параметрами.

5. При изменении параметра качества текстур, новые настройки сохраняются, а меню перерисовывается с новыми параметрами.

6. При нажатии клавиши Esc игра возвращается в главное меню.

7. Вариант использования завершается.

##Просмотр статистики

Данный вариант использования позволяет пользователю просмотреть свою статистику
Предусловие: игра находится в состоянии ожидания нажатия одной из клавиш в главном меню
Основной поток событий
 
1. Вариант использования начинается, когда пользователь нажимает кнопку Stats в главном меню.

2. Главное меню сменяется на экран статистики.

3. При нажатии клавиши Esc игра возвращается в главное меню.

4. Вариант использования завершается.
